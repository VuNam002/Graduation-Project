🆘 Troubleshooting Guide (Click to expand)

❌ Lỗi: API Connection Failed

  • Kiểm tra backend đã chạy chưa: dotnet run
  • Kiểm tra port: Mở launchSettings.json xem port là bao nhiêu
  • Nếu dùng HTTPS, vào trình duyệt https://localhost:7215 và accept certificate
  • Thử đổi sang HTTP trong dropdown "Protocol"

❌ Lỗi: WebSocket Connection Failed

  • Quan trọng: Backend PHẢI có WebSocket endpoint. Check file Program.cs
  • Cần thêm code sau vào Program.cs:
app.UseWebSockets(); app.Map("/ws", async context => { if (context.WebSockets.IsWebSocketRequest) { var webSocket = await context.WebSockets.AcceptWebSocketAsync(); var wsManager = context.RequestServices.GetRequiredService<WebSocketManagerService>(); await wsManager.HandleWebSocketAsync(webSocket); } });

❌ Lỗi: Camera Cannot Open

  • Camera ID có thể khác 0, thử các giá trị: 0, 1, 2
  • Kiểm tra camera đã được app khác sử dụng chưa (Teams, Zoom, Skype...)
  • Kiểm tra quyền truy cập camera trong Windows Settings
  • Thử chạy app với quyền Administrator

❌ Lỗi: No Frames Received

  • Backend phải GỬI frames qua WebSocket (hiện tại code chưa có phần này)
  • Cần thêm logic streaming trong ProcessCameraFeed
  • Xem phần "Missing Backend Code" bên dưới

✅ Nếu mọi thứ OK nhưng vẫn lỗi

  • Mở Developer Console (F12) để xem lỗi chi tiết
  • Check Network tab để xem API requests
  • Check Console tab để xem JavaScript errors
  • Gửi screenshots của logs cho dev team
